MuseGAN

Multi-track Sequential Generative Adversarial Networks for Symbolic Music Generation and Accompaniment.

MuseGAN is a project on music generation. In essence, we aim to generate polyphonic music of multiple tracks (instruments) with harmonic and rhythmic structure, multi-track interdependency and temporal structure. Details and demos on our Github page!

Check out the paper.

Neural Wavetable Synthesizer

A project with the legendary Lamtharn (Hanoi) Hantrakul

A wavetable synthesizer plugin that uses WaveNet to interpolate between different wavetables. We include two formats of this synthesizer, a plug-and-play VST you can download and play RIGHT NOW in your DAW! For the more technically inclined, we also include all our source python and c++ code using tensorflow and JUCE for you to experiment and build upon. Happy music-making!

Please find the details and download description in the Demo Video and GitHub repo.

MGEval

On the evaluation of generative models in music.

The modeling of artificial, human-leve l creativity is becoming more and more achievable. In recent years, neural networks have been successfully applied to different task s such as image and music generation, demonstrating their great potential in realizing computational creativity. This project propose a set of simple musically informed objective metrics enabling an objective and reproducible way of evaluating and comparing the output of music generative systems. We demonstrate the usefulness of the proposed metrics with several experiments on real-world data.

The work is published as "On the evaluation of generative models in music" and released as an open-source toolbox .
